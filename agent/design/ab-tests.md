## How to Run an A/B Test

1. Document hypothesis in `Active Tests` table above
2. Create a branch: `agent/design/ab-test-[name]`
3. Implement the variant
4. Commit with test ID in message
5. Merge when ready
6. Move to `Completed Tests` table with result and decision
